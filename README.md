## 配置说明

1、修改项目配置文件 nav.config.ts 只需要修改仓库地址

2、https://github.com/settings/tokens 申请 token, 勾选相应的权限(repo, user), 如果不懂就全部选中，复制并保存 Token；

3、https://github.com/anaer/nav/settings/secrets/actions/new 添加申请的 token， name 填写 TOKEN 大写。

4、https://github.com/anaer/nav/actions 开启 action 自动部署

5、https://anaer.github.io/nav 查看导航网站

6、https://anaer.github.io/nav/#/system 后台管理

## 文件说明

| 文件名             | 说明         |
| ------------------ | ------------ |
| nav.config.ts      | 项目配置文件 |
| data/settings.json | 站点配置信息 |
